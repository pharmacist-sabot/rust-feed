## หลักการสำคัญของหมวด Rust Research

หมวด Rust Research มุ่งเน้นการย่อยงานวิจัยเชิงวิชาการ (Academic Research), Experimental Features, Formal Verification, หรือแนวคิดทางทฤษฎีที่เกี่ยวข้องกับ Rust ให้อ่านง่ายและนำไปต่อยอดได้ โดยเน้นการอธิบายว่า "ทำไมงานวิจัยนี้ถึงสำคัญต่อ Rust Community" และ "มันจะส่งผลกระทบอย่างไรในอนาคต"

---

## แนวทางการเลือกหัวข้อ

- งานวิจัยจาก Academic Papers ที่เกี่ยวข้องกับ Rust (เช่น Memory Safety Proofs, Ownership Model Analysis)
- Formal Verification ของ Rust Programs (เช่น Prusti, Creusot, Kani, Verus)
- Experimental Language Features ที่ยังอยู่ในขั้น RFC หรือ Nightly
- การวิเคราะห์เชิงเปรียบเทียบระหว่าง Rust กับภาษาอื่นในเชิง Theory
- Rust ในบริบทของ Programming Language Theory (PLT)
- Performance Benchmarking เชิงลึกที่มี Methodology ชัดเจน
- แนวคิดจาก Type Theory, Category Theory, Linear Types ที่สะท้อนใน Rust
