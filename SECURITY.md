# Security Policy

## Supported Versions

Since this project (Rust Feed) is primarily a curated content repository and markdown book using `mdBook`, rather than a software library or application, security updates are generally applied only when underlying dependencies (like `mdBook` itself or the GitHub Actions used for deployment) require updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project (e.g., related to the build scripts, GitHub Actions workflows, or potential XSS in the markdown content rendering), please do not report it through public GitHub issues.

Instead, please send an email to the project maintainer directly or report it privately through GitHub's vulnerability reporting feature (if enabled).

We will try to acknowledge and address the issue as quickly as possible.
